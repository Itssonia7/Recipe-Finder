# 🍽️ Recipe Finder

A modern, responsive Recipe Finder web application built using **HTML, CSS and JavaScript**.

The application allows users to search real recipes using **TheMealDB API**, save favorite recipes using Local Storage, switch between Light and Dark mode, and view recipes in a clean responsive interface.

---

## ✨ Features

* 🔍 Search recipes
* 🌍 Real-time Recipe API
* ❤️ Save Favorites
* 💾 Local Storage
* 🌙 Dark Mode
* 📱 Responsive Design
* 🍞 Toast Notifications
* 🍽️ View Recipe Source

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* TheMealDB API
* Local Storage



---

## 🚀 Installation

```bash
git clone https://github.com/Itssonia7/recipe-finder.git

cd recipe-finder
```

Open

```
index.html
```

in your browser.
